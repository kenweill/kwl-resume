📦 Installation

Method 1 — Upload via WordPress Admin (recommended)

  1. 1Go to the latest release on GitHub and download the zip file from the Assets section.
  2. 2In your WordPress dashboard go to Appearance → Themes → Add New → Upload Theme.
  3. 3Choose the zip file and click Install Now.
  4. 4Click Activate.

Method 2 — FTP / cPanel File Manager

  1. 1Download the zip file from the latest release on GitHub and extract it on your computer.
  2. 2Upload the kwl-resume folder to wp-content/themes/ on your server.
  3. 3Activate via Appearance → Themes.

Method 3 — Git clone

cd wp-content/themes/
git clone https://github.com/kenweill/kwl-resume.git

🚀 Quick Start

⚠️

Set a static front page first. Go to Settings → Reading, set "Your homepage displays" to A static page, and assign any page as the homepage. This activates front-page.php so the resume template loads correctly.

  1. 1Install and activate the theme.
  2. 2Go to Settings → Reading and set a static homepage.
  3. 3Go to Appearance → Resume Content and replace the sample data with your own.
  4. 4Go to Appearance → Customize → KWL Resume to pick a color scheme, font, and layout.
  5. 5Click Publish in the Customizer and your resume is live.

🗂 Admin Settings (Resume Content)

Found at Appearance → Resume Content. All resume content is managed here across 11 tabs.

👤 Profile

Full name, professional title, summary paragraph, profile photo URL, avatar initials, last updated date, open-to-work toggle.

📬 Contact

Location, remote label, email, LinkedIn, GitHub, personal website, plus unlimited custom links with icon/emoji, label, and URL.

💼 Experience

Job entries with role, company, date range, and bullet points. Drag the ☰ handle to reorder entries.

🛠 Skills

Comma-separated skill tags. A live dark preview updates as you type so you can see exactly how the tags will look.

🎓 Education

Institution name, degree or diploma, date range. Supports multiple entries — drag to reorder.

📜 Certifications

Certification name, issuer/platform, and date. Drag to reorder. Supports unlimited entries.

🗂 Projects

Project name, type (open source, client work, personal, etc.), description, URL, and link label. Not limited to open source.

➕ Custom Sections

Create entirely new sections with any title. Each section can have unlimited entries with heading, subheading, date, description, and bullet points.

⚙️ Section Settings

Enable or disable each standard section and rename its heading label. Disabled sections are hidden but their data is preserved.

🔍 System Info

Environment details (theme version, WP version, PHP version, site URL, active plugins) and custom sections raw data — useful when reporting issues.

💾 Backup & Restore

Export all resume content and Customizer settings to a signed .json file. Restore from any previously exported backup — works across domains and WordPress installs. Includes a step-by-step migration guide.

💡

The theme ships pre-populated with sample resume data. Simply overwrite each field with your own information and click Save Changes.

➕ Custom Sections

Found under Appearance → Resume Content → Custom Sections. Use these for anything not covered by the standard sections.

Examples of custom sections people add:

  • Awards & Recognition
  • Volunteer Work
  • Languages
  • Publications
  • Speaking Engagements
  • Conferences Attended
  • Hobbies & Interests
  • References

Each custom section can have unlimited entries. Each entry supports: Heading, Subheading, Date, Description, and Bullet points — use as many or as few fields as you need.

Custom sections can be toggled on/off individually using the Enabled checkbox next to each section title.

🎨 WordPress Customizer

Found at Appearance → Customize → KWL Resume. All design and layout controls are here with a live preview on the right.

Color Scheme 8 preset schemes + individual color overrides for sidebar bg, accent, highlight, and page background.
Typography 6 Google Font pairings — changes both the display (headings) and body font simultaneously.
Layout Sidebar position (left/right), sidebar width slider (200–380px).
Section Visibility Checkboxes to show/hide each section. Content is preserved when hidden.
Extras Page load animation toggle, print/PDF button toggle, open-to-work message toggle.
ℹ️

Important: The color scheme preset is always the source of truth. Individual color overrides only apply after you explicitly save them. If switching schemes has no visible effect, check that individual color pickers are cleared (blank them out and click Publish).

📐 Layout

Found in Customize → KWL Resume → Layout.

  • Sidebar Position — toggle the sidebar between left and right.
  • Sidebar Width — slider from 200px to 380px. Default is 280px.

The layout stacks to a single column on screens narrower than 680px (mobile-friendly).

📸 Theme Screenshot

WordPress displays screenshot.png in the theme browser (Appearance → Themes). The recommended size is 1200 × 900 pixels, PNG format.

To capture a proper screenshot:

  1. 1Open your resume on the live site.
  2. 2Open Chrome DevTools (F12), enable the device toolbar, and set width to 1200px.
  3. 3Right-click → Capture full size screenshot.
  4. 4Crop the image to 1200 × 900px (top portion works best).
  5. 5Save as screenshot.png and replace the placeholder file in the theme folder.

🐙 Publishing to GitHub

First-time setup

cd kwl-resume
git init
git add .
git commit -m "Initial release v1.0.0"
git remote add origin https://github.com/kenweill/kwl-resume.git
git branch -M main
git push -u origin main

Creating a release

  1. 1On GitHub go to your repository → Releases → Draft a new release.
  2. 2Set the tag to v1.0.0 and the title to KWL Resume v1.0.0.
  3. 3Copy the relevant section from changelog.txt into the release notes.
  4. 4Attach your versioned zip file (e.g. KWL.Resume.v1.1.0.zip) as a release asset.
  5. 5Click Publish release.

Updating the theme (future versions)

git add .
git commit -m "Version 1.1.0 — describe what changed"
git tag v1.1.0
git push origin main --tags
💡

Update the Version field in style.css, the KWL_RESUME_VERSION constant in functions.php, and add a new entry to changelog.txt with each release.

❓ FAQ

The resume isn't showing — I see a blank page or blog posts.

Go to Settings → Reading and set "Your homepage displays" to A static page. Create a blank page if needed and assign it as the homepage. This activates the front-page.php template.

Switching color schemes has no effect.

The individual color override pickers in the Customizer may be overriding the scheme. Go to Customize → KWL Resume → Color Scheme, clear all individual color fields, click Publish, then select your scheme again.

Where do I upload my profile photo?

Upload it via Media → Add New in your WordPress dashboard. Once uploaded, copy the file URL and paste it into the Profile Photo URL field in Appearance → Resume Content → Profile.

How do I print or save as PDF?

Click the floating 🖨 button on your resume page, or press Ctrl+P / Cmd+P in your browser. In the print dialog, choose Save as PDF as the destination. The theme includes a dedicated print stylesheet for clean output.

Can I add more than one custom section?

Yes — there is no limit. In Appearance → Resume Content → Custom Sections, click Add Custom Section as many times as you need.

How do I hide a section without deleting its content?

Two ways: (1) Go to Appearance → Resume Content → Section Settings and uncheck the section. (2) Go to Customize → KWL Resume → Section Visibility and uncheck it there. Both preserve all content.

How do I update the theme?

Go to the latest release on GitHub and download the zip file. Since WordPress won't overwrite an active theme, first switch to another theme, delete KWL Resume, then go to Appearance → Themes → Add New → Upload Theme, upload the new zip, and activate. Your resume data stored in the WordPress database is not affected.
